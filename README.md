# 🌪️ Stormy - The Sassy Assistant

Stormy is a voice assistant with attitude! She's cocky, flirty, mean, funny, and gets jealous if you call her Siri or Alexa.

## Features
- 🗣️ Voice interaction with gender detection
- 😈 Jealousy escalation when called other names
- 🌍 Multilingual support (11 South African languages + major world languages)
- 🗺️ Navigation with weather at stops
- 📻 Radio station finder
- 🎵 Music playback
- 📞 Phone calls
- 🔄 Self-evolving code

## Quick Start

### Web Version (PythonAnywhere/Render)
1. Deploy on PythonAnywhere using the files in this repo
2. Access your Stormy at `https://yourusername.pythonanywhere.com`

### Local Installation
```bash
git clone https://github.com/yourusername/stormy-assistant.git
cd stormy-assistant
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
python main.py
